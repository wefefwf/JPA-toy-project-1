#  나의 포근한 할 일 일기 (Warm Todo-Diary)

> **"지나간 일도, 다가올 일도 따뜻하게 기록하는 공간"** > Spring Boot와 JPA를 기반으로 제작된, 다이어리 감성의 할 일 관리 서비스입니다.

---

##  Project Concept
기존의 딱딱한 할 일 목록에서 벗어나, 실제 **다이어리 속지**를 채워 나가는 듯한 포근하고 부드러운 분위기를 지향합니다.



* **Color Palette:** Cream Ivory (`#FFFDF5`), Sage Green (`#A8BA9A`), Soft Rose (`#E59A9A`)
* **Design:** 둥근 모서리와 부드러운 그림자(Soft Shadow)를 활용한 카드 UI 적용

---

##  Key Features (주요 기능)
본 프로젝트는 JPA를 활용한 기본적인 **CRUD** 기능을 충실히 구현하고 있습니다.

1. **메인 목록 (List)**
   - 등록된 모든 할 일을 감성적인 카드 형태로 확인합니다.
   - 마우스 호버(Hover) 시 부드러운 애니메이션 효과가 적용됩니다.
2. **기록 열람 (Read)**
   - 특정 할 일의 상세 내용을 다이어리 속지 레이아웃으로 확인합니다.
3. **새 조각 남기기 (Register)**
   - 새로운 할 일과 마감 기한을 직관적인 폼을 통해 기록합니다.
4. **기록 다듬기 (Modify & Delete)**
   - 작성된 내용을 수정하거나, 필요 시 부드러운 확인 절차를 거쳐 삭제합니다.
