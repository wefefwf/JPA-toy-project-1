#  TodoList — 모눈종이 위의 하루 (Grid Paper Daily)

> **"민트빛 캘린더 위에, 오늘 할 일을 하나씩 채워가는 공간"**

> Spring Boot와 JPA를 기반으로 제작된, 캘린더 앱 감성의 할 일 관리 서비스입니다.

---

##  Project Concept

캘린더 앱에서 영감을 받아, **민트 그린 배경 위에 글래스모피즘 카드**가 올려진 깔끔하고 산뜻한 UI를 지향합니다. 각 페이지마다 다른 배경 패턴을 적용해 화면 전환 시 시각적 리듬감을 줍니다.

* **Color Palette:** Mint Green (`#C5D8B5`), Coral Red (`#E04A3C`), Clean White (`#FFFFFF`)
* **Typography:** Outfit — 깔끔한 산세리프 서체로 가독성과 모던함을 동시에
* **Design:** 반투명 글래스모피즘 카드, 모눈종이 격자 배경, 페이지별 컬러 시스템

---

##  Page Design System

| 페이지 | 배경 | 특징 |
|--------|------|------|
| **List** | 🟢 민트 그린 그라데이션 | 현재 달 캘린더 + 글래스 카드 목록 |
| **Register** | 🟢 민트 그린 + 흰색 격자 | 모눈종이 위의 입력 폼 |
| **Read** | ⬜ 흰색 + 민트 반원 블러 | 상세 내용 가운데 정렬, 스머징 배경 |
| **Modify** | ⬜ 흰색 + 빨간 격자 | 수정 모드를 시각적으로 구분 |

---

##  Key Features (주요 기능)

본 프로젝트는 JPA를 활용한 기본적인 **CRUD** 기능을 충실히 구현하고 있습니다.

1. **메인 목록 + 캘린더 (List)**
   - 현재 월 캘린더가 상단에 표시되며, 오늘 날짜는 코랄 레드로 강조됩니다.
   - 월 이동(◀ ▶) 버튼으로 다른 달도 확인할 수 있습니다.
   - 등록된 할 일을 반투명 글래스 카드로 확인하며, 호버 시 부드러운 애니메이션이 적용됩니다.

2. **기록 열람 (Read)**
   - 민트 블러 반원 배경 위 흰 카드에 할 일 상세 내용을 표시합니다.
   - 내용과 마감일이 가운데 정렬되어 한눈에 파악 가능합니다.

3. **새 기록 남기기 (Register)**
   - 민트 모눈종이 배경 위에서 새로운 할 일과 마감 기한을 입력합니다.
   - 입력 포커스 시 민트 테두리로 시각 피드백을 제공합니다.

4. **기록 다듬기 (Modify & Delete)**
   - 빨간 격자 배경으로 수정 모드임을 직관적으로 구분합니다.
   - 삭제 시 확인 대화상자를 통해 실수를 방지합니다.
